01. Logo System

Gravity Arcs / 引力弧线

Gravity Arcs 是 UNBOUNDX 的核心视觉图腾。回到最本质的两条弧线：一条象征活力、人文与向前引力的橙色轨迹；一条象征精准、科技与秩序的冷白弧线。两者在深色空间中相交，形成向右上方突破的极简 X，寓意进入全新的独立维度。

Geometry Reset

极简形态

以最纯粹的几何形态构建品牌视觉，摒弃一切冗余装饰，让标志回归至极简、纯粹且极具辨识度的终极状态。

Dimension

X 代表维度

在这里，X 不是普通交叉符号，而是品牌语义里的 Dimension。它象征独立维度、自由空间，以及进入新秩序的入口。

Gravity Arcs

引力牵引，向上突破

两条弧线像被看不见的引力场牵引，最终在右上方向形成爆发点。这个运动方向本身就在讲“超越常规”的故事。

Freedom

自由维度的情绪包装

它不是冷冰冰的科技图形，而是有态度的精神符号：自由、不受限、独立、活出自我，同时依然保有现代 SaaS 的克制与高级感。

Primary Lockup

横向标准组合。适用于网站导航栏、文档页眉、名片等绝大多数宽幅场景。

Stacked Lockup

纵向居中组合。适用于 App 启动页、海报中心、社交媒体封面等需要视觉聚焦的场景。

Icon / Avatar

纯图形标识。高度凝练，即使缩小至 16x16 px 作为网页 Favicon，依旧清晰锐利。

Light Mode Reversed

当必须在白底环境（如打印纸张、第三方平台）使用时，白色弧线和字体反转为黑曜石色。

Why we chose Aerospace Orange
01 · 品牌心理学

航天橙在硬核科技与人文温度之间取得完美平衡，赋予品牌破局者的锐利气质与创新活力，构建极具能量的视觉语言。

02 · Dark Mode 统治力

在 Deep Void 背景上，#F26122 具备极高视觉信噪比，能在复杂数据界面中迅速聚焦用户注意力，构建清晰的视觉层级。

03 · 跨媒介稳定性

精准设定的色彩参数，确保线上网页、UI 界面与线下印刷物料的色彩呈现保持高度统一，维持品牌视觉的绝对一致性。

总结来说，Aerospace Orange 不只是一个主色，而是 UNBOUNDX “在未知维度中，提供清晰、安全且充满活力的指引”的视觉图腾。

04. UI Guidelines

Digital Product Aesthetics

UNBOUNDX 的 UI 设计必须贯彻「自由维度」的精神：摒弃繁复装饰，拒绝平庸模板，以极简的线条、深邃的空间与克制的动效，打造出具备“破局者”气质的硬核产品界面。

RULE 01
克制即力量 / Restrained Power
彻底抛弃多余的背景色块、无意义的投影与渐变。使用极细的 1px 描边 (var(--surface-2)) 来划分层级。让信息本身成为界面的主角，而非容器。
RULE 02
深色沉浸 / Dark Immersion
绝不使用纯黑。基于 #050505 (Deep Void) 构建深色背景，利用 #101012 与 #1A1A1E 构建微弱的表面纵深。文字与元素必须在此环境中保持锐利与高对比。
RULE 03
张力排版 / Tension Typography
西文数字必须使用 Outfit，利用其几何张力构建大标题与核心数据。打破常规的等距网格，通过留白（Negative Space）与不对称排版创造阅读节奏。
