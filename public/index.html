01. 标志系统

Gravity Arcs / 引力弧线

两条弧线在深色空间中相交，形成向右上方突破的极简 X——橙色象征活力与向前引力，冷白代表精准与秩序，寓意进入全新的独立维度。

标准横版

适用于网站导航栏、文档页眉、名片等绝大多数宽幅场景。

纵向组合

适用于 App 启动页、海报中心、社交媒体封面等需要视觉聚焦的场景。

图标 / 头像

高度凝练，即使缩小至 16x16 px 作为网页 Favicon，依旧清晰锐利。

浅色反转

当必须在白底环境（如打印纸张、第三方平台）使用时，白色弧线和字体反转为黑曜石色。

为什么选择航天橙？
01 · 品牌心理学

航天橙在硬核科技与人文温度之间取得完美平衡，赋予品牌破局者的锐利气质与创新活力，构建极具能量的视觉语言。

02 · Dark Mode 统治力

在 Deep Void 背景上，#F26122 具备极高视觉信噪比，能在复杂数据界面中迅速聚焦用户注意力，构建清晰的视觉层级。

03 · 跨媒介稳定性

精准设定的色彩参数，确保线上网页、UI 界面与线下印刷物料的色彩呈现保持高度统一，维持品牌视觉的绝对一致性。

总结来说，Aerospace Orange 不只是一个主色，而是 UNBOUNDX “在未知维度中，提供清晰、安全且充满活力的指引”的视觉图腾。
标志理念
Gravity Arcs 设计哲学
Geometry Reset · 极简形态

以最纯粹的几何形态构建品牌视觉，摒弃一切冗余装饰，让标志回归至极简、纯粹且极具辨识度的终极状态。

Dimension · X 代表维度

在这里，X 不是普通交叉符号，而是品牌语义里的 Dimension。它象征独立维度、自由空间，以及进入新秩序的入口。

Gravity Arcs · 引力牵引，向上突破

两条弧线像被看不见的引力场牵引，最终在右上方向形成爆发点。这个运动方向本身就在讲”超越常规”的故事。

Freedom · 自由维度的情绪包装

它不是冷冰冰的科技图形，而是有态度的精神符号：自由、不受限、独立、活出自我，同时依然保有现代 SaaS 的克制与高级感。

04. 界面规范

数字产品美学

摒弃繁复装饰，拒绝平庸模板，以极简的线条、深邃的空间与克制的动效，打造出具备”破局者”气质的硬核产品界面。

RULE 01
克制即力量 / Restrained Power
抛弃多余的背景色块、投影与渐变。使用极细的 1px 描边划分层级，让信息成为界面主角。
RULE 02
深色沉浸 / Dark Immersion
基于 #050505 (Deep Void) 构建深色背景，利用 #101012 与 #1A1A1E 构建微弱的表面纵深。
RULE 03
张力排版 / Tension Typography
西文数字使用 Outfit，利用其几何张力构建大标题与核心数据。通过留白与不对称排版创造阅读节奏。

06. 动效系统

动效即品牌行为

四层动效体系：品牌入场（开屏/片头）、加载等待（处理态）、交互响应（悬停/点击）、氛围层（背景营造）。

您的浏览器不支持 Canvas 动画
Motion System Architecture

四层动效体系，覆盖品牌到产品的完整场景

从 品牌入场、加载等待、交互响应、氛围营造 四个维度构建动效语言。

Brand Entrance Micro Loaders Interactive Response Ambient Layer
Layer 01 · Brand Entrance

品牌入场：开屏、片头、首次加载

弧线绘制、粒子爆发、扫光揭示等高仪式感动效，用于品牌首次出现的关键时刻。

Layer 02 · Micro Loaders

加载等待：处理中、生成中、同步中

旋转加载器、脉冲呼吸、进度指示等克制的等待反馈。

Layer 03 · Interactive Response

交互响应：悬停、点击、状态切换

发光脉冲、磁场吸引、弹性反馈等微交互动效，为按钮、卡片、导航提供即时反馈。

Layer 04 · Ambient Layer

氛围营造：背景、空间、持续运转

轨道拖尾、磁场线条、扫描网格等环境动效，为页面注入空间深度和品牌气质。
